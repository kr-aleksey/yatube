Технологии проекта

  • Python
  • Django
  • SQL
  • HTML и CSS

Python – современный язык программирования, работающий на всех распространенных операционных системах. Этот язык очень гибкий и мощный. Синтаксис Python интуитивно понятен. Он подходит как для первого знакомства с программированием, так и для реализации больших коммерческих проектов. Для меня изучение программирования на других языках никогда не было таким интересным и понятным как на Python. Python – это язык на котором написан backend этого сайта. В начале изучения Python мне очень помогла бесплатная часть курса Яндекс Практикума “Python Разработчик”.

Django framework – это технология, которая берет на себя всю рутинную работу web разработчика. Django – каркас этого сайта. Я изучаю Django в группе на Яндекс Практикум. И это моя новая ступень в окружении интересных людей. Нам помогает профессионалы Практикума – куратор, наставники, код-ревьюеры.

SQL – язык запросов реляционных баз данных. Сейчас ни один сайт, online приложение не обходится без SQL.

HTML и CSS – это frontend любого сайта. Этот сайт не исключение. Эти технологии я изучал на курсе freeCodeCamp “Responsive Web Design”. Я написал итоговый проект и получил сертификат.
